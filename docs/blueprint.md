# **App Name**: StockSight

## Core Features:

- Real-time Inventory Display: Display real-time data from IoT sensors on a unified dashboard.
- Inventory Status Table: Display an interactive table with key inventory metrics (material type, quality, weight, height, status).
- Threshold Triggering: Alert the user when weight/height metrics go below defined material thresholds
- Trend Visualization: Generate graphs and charts for visualizing inventory trends over time.
- Report Generation: Generate and export inventory reports in PDF or Excel formats.
- Predictive Reordering: Use generative AI tool that suggests optimized order schedules based on past data and anticipated use of materials. 
- Role-Based Access Control: Implement role-based access control (staff, manager, admin, technician) to manage UI element visibility.

## Style Guidelines:

- Primary color: Navy blue (#002B5B) to reflect the reliable, professional nature of a supply chain application.
- Background color: Light gray (#F9F9F9) for light mode.
- Accent color: Sky blue (#00B4D8) to draw attention to key metrics.
- Body font: 'PT Sans' (sans-serif) for body text. Headline font: 'Space Grotesk' (sans-serif) for a modern, technical look.
- Use monochrome icons from Lucide or Material Icons, white on dark, navy on light.
- Desktop-first responsive design using a 12-column grid layout.
- Subtle transitions and animations using Framer Motion or CSS transitions for smooth user experience.
